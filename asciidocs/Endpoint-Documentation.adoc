=== Start the Endpoint Tests

Before you start, make sure that you have installed Docker.

First you have to start the Database with following command. You can find the docker-compose.db.yml File in the backend Folder.

[source, console]
----
  docker-compose  -f docker-compose.db.yml up
----

Now you have to start Quarkus with this command:

[source, console]
----
  ( For Linux "./")mvnw clean complile quarkus:dev
----

Now you can execute the karate Tests at the "Play" symbol
